
# Clash Manager

**Clash Manager: Clan Manager for Clash Royale** is an advanced, serverless PWA built on **Google Apps Script** and **React**.

Unlike simple API wrappers, Clash Manager offers **stateful database logging**, **Moneyball-style performance scoring**, and an **automated headhunter** that actively scouts talent in the background.

## Features

- **Moneyball Scoring**
- **Automated Headhunter**
- **Persistent Database**
- **Serverless PWA**
  
## Tech Stack

- **Frontend**: React 19, TypeScript, Lucide React, Tailwind CSS
- **Backend**: Google Apps Script (V8 Runtime)
- **Data**: Google Sheets
- **API**: RoyaleAPI Proxy
